# 示例文档目录

本目录包含5份与"自然语言处理技术"相关的TXT文档，用于测试RAG问答系统。

## 文档列表

1. **1_自然语言处理简介.txt** - NLP的基本概念、发展历程和主要任务
2. **2_词嵌入技术详解.txt** - 词嵌入的概念、经典模型和应用
3. **3_Transformer模型原理.txt** - Transformer架构、自注意力机制详解
4. **4_BERT模型详解.txt** - BERT模型的预训练、微调和应用
5. **5_文本分类技术综述.txt** - 文本分类的传统方法和深度学习方法

## 说明

这些文档为TXT格式，可直接被系统加载。如需测试PDF或DOCX格式，
可将这些内容转换为对应的格式后放入本目录。
